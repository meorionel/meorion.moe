### **Top-up Bachelor's Degree Exam**

In 2024 summer holiday, my original plan was to find an internship job, but I couldn't find any job in Nanning.

Later, I found out that my school was offering free training courses for the Top-up Bachelor's Degree entrance exam.

Surprisingly, the 2025 exam turned out to be much easier than I had expected. In the end, I was admitted to a bachelor's degree program. Although it is not a top-tier university, it is still a bachelor's degree ‚Äî and for me, that matters.

## What projects have you made üì¶

### **PinkLine**

https://github.com/meorionel/PinkLine

PinkLine is a blog template I built at the beginning of this year. Since I started blogging in 2023, I've gone through several blog themes, all of which were developed by myself. My tech stack evolved from Hexo to VitePress, then to Vue with MySQL after I learned Vue.

Whenever I learn a new skill, I tend to apply it directly to my blog. As my skills improve, I often feel that my earlier code has become messy or overcomplicated, so I refactor it repeatedly. (GPT even awarded me "The Infinite Refactor Award")

This version of the blog was developed after I became familiar with Next.js and Nuxt. To be honest, Nuxt is excellent for content-driven websites. It provides out-of-the-box SSR, auto imports, file-based routing, and has a strong community ecosystem. After getting used to it, I now find it difficult to go back to plain Vue.

My previous blog was built with Vue and MySQL, which made content management unnecessarily abstract and complex. In contrast, PinkLine uses Nuxt's official CMS (nuxt/content), which significantly simplifies the entire workflow.

All articles are written as plain .md files, even pages. Nuxt also provides an official content editor ‚Äî you only need to install the dependencies to start using it.

### **Orange**

https://github.com/meorionel/orange

During the summer holiday, I started a Minecraft server with my friends. As usual, we needed a gameplay document to help new players get started more easily.

Other servers typically use online tools such as Tencent Docs. At first, others planned to do the same, but since this was closely related to my professional field, I insisted on building an official website from scratch.

The site is also built with nuxt/content. In addition, I implemented several backend interfaces using Nuxt's built-in server to query and display server plugin data.

### **nuxt-oneko**

https://github.com/meorionel/nuxt-oneko

nuxt-oneko is a Nuxt plugin that displays a kitten on the page, inspired by the classic Macintosh desktop toy. The kitten follows the user's pointer and moves around dynamically.

The original version was a standalone JavaScript plugin. I refactored it into a Nuxt plugin, allowing developers to use it simply by installing the package and adding it to nuxt.config.ts.

### **Minecraft3DHead**

https://github.com/meorionel/Minecraft3DHead

This interface is written in js and can obtain the player's Minecraft skin, then crop and stitch it together to form a 3D head. It is an interface used in Orange documentation.

### **Audio**

This year, I bought two pairs of earphones, both from Moondrop ‚Äî that's what a loyal Moondrop fan would say, after all.
